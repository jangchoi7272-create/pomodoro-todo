# 🍅 포모도로 + 할일 (Pomodoro + Todo)

집중 시간 관리를 위한 포모도로 타이머와 할 일 목록을 한 화면에서 사용할 수 있는 간단한 웹 앱입니다.

## 기능

- **포모도로 타이머**
  - 집중 25분 / 짧은 휴식 5분 / 긴 휴식 15분 모드 전환
  - 시작 / 일시정지 / 리셋
  - 타이머 종료 시 알림음 재생 및 알림 메시지 표시
  - 완료한 집중 세션 횟수를 로컬에 저장하여 누적 표시
- **할 일 목록**
  - 할 일 추가
  - 클릭하여 완료 처리 (취소선 표시)
  - 삭제
  - 브라우저 `localStorage`에 저장되어 새로고침해도 유지됨

## 실행 방법

별도의 빌드나 설치 과정 없이 `index.html` 파일을 브라우저에서 열기만 하면 됩니다.

```bash
# 예: 기본 브라우저로 열기 (Windows)
start index.html
```

또는 VS Code의 Live Server 같은 확장으로 로컬 서버를 띄워 실행해도 됩니다.

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 화면 구조(타이머, 할 일 목록) |
| `style.css` | 전체 스타일 |
| `script.js` | 타이머 로직 및 할 일 목록 로직 |

## 기술 스택

순수 HTML / CSS / JavaScript (별도 프레임워크나 의존성 없음)
